# 📊 YNM HTML Reports Dashboard

A premium dashboard to archive and view team HTML reports directly on GitHub Pages.

## 🚀 How to "Upload" Reports

To add a new report, simply:
1.  **Drop your HTML file** into the `reports/` folder.
2.  **Commit and Push** your changes to GitHub.
3.  The **GitHub Actions** will automatically:
    *   Update the manifest (`manifest.json`)
    *   Deploy the latest reports to your dashboard.

## ⚙️ Initial Setup for GitHub Pages

1.  **Create a New Repository** on GitHub.
2.  **Upload this code** to that repository.
3.  Go to **Settings** > **Pages**.
4.  Under **Build and deployment** > **Source**, select **GitHub Actions**.
5.  Wait a few minutes, and your reports will be live at `https://<your-username>.github.io/<repo-name>/`.

*   **HTML/CSS/JS**: Clean, modern, responsive design.
*   **Lucide Icons**: Beautiful visual indicators.
*   **Glassmorphism**: Elegant translucent UI components.
*   **GitHub Actions**: Continuous deployment and manifest automation.
